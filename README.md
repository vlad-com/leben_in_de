# leben_in_de
This website has been created to help you prepare for the "Leben in Deutschland" Test and "Einbürgerungstest".

All questions and images on this website are taken from the [official catalog](https://www.bamf.de/SharedDocs/Anlagen/DE/Integration/Einbuergerung/gesamtfragenkatalog-lebenindeutschland.pdf?__blob=publicationFile&v=23):
General Catalog of Questions for the "Leben in Deutschland" Test and "Einbürgerungstest" (Stand: 07.05.2025).

The answers have been additionally double-checked manually using the official BAMF resource:
[https://oet.bamf.de/ords/oetut/f?p=514:1
